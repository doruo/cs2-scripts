# Counter-Strike 2 Scripts

## Description

Counter-Strike 2 configuration scripts for startup and practice. Made for personnal purpose. 

## Add scripts in your game

### Access game files

- On Steam platform, right click on your CS2 game -> Manage -> Browse local files.

- Or to access directly in local files explorer: SteamLibrary/steamapps/common/Counter-Strike Global Offensive/.

### Add config files

- Then add cfg files in directory game/csgo/cfg.  

## Usage

- On a custom game, type in your console: 

```
exec [filename].cfg
```

- Example usage:

```
exec practice.cfg
```

```
exec custom.cfg
```
